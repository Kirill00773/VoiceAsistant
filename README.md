# 🎤 Локальный голосовой ассистент для студентов лаборатории

## 📌 Описание проекта

Данный проект представляет собой локального голосового ассистента, разработанного для помощи студентам лаборатории.

Ассистент работает полностью оффлайн и использует локальную большую языковую модель (LLM) для ответа на вопросы на основе базы знаний лаборатории.

Основная задача проекта — упростить доступ к информации для новых участников лаборатории, таких как:

* правила работы в лаборатории
* инструкции
* FAQ
* организационные вопросы

---

## 🧠 Возможности

* 📚 Поиск информации в базе знаний (RAG)
* 🤖 Генерация ответов с помощью локальной LLM (Llama 3 через Ollama)
* 🎤 Распознавание речи (Speech-to-Text)
* 🔊 Озвучивание ответов (Text-to-Speech)
* 💬 Интерактивный режим общения

---

## ⚙️ Используемые технологии

* Python
* Ollama (локальная LLM)
* Llama 3
* LangChain
* ChromaDB
* Faster-Whisper (STT)
* Silero TTS

## 🚀 Как запустить проект

### 1. Клонирование проекта

git clone <https://github.com/Kirill00773/VoiceAsistant.git>

cd voice_assistant

---

### 2. Создание виртуального окружения 

python -m venv .venv
.venv\Scripts\activate   

---

### 3. Установка зависимостей

pip install -r requirements.txt

Если файла requirements.txt нет, установите вручную:

pip install langchain langchain-chroma langchain-ollama chromadb
pip install faster-whisper sounddevice scipy
pip install torch omegaconf

---

### 4. Установка и запуск Ollama

Скачайте и установите Ollama:

https://ollama.com/

Затем выполните:

ollama pull llama3
ollama run llama3

---

### 5. Подготовка базы знаний

Поместите текстовые файлы (.txt) в папку:

data/knowledge_base/

---

### 6. Запуск текстового режима

python init_db.py

Введите вопрос в консоль — ассистент найдёт информацию и ответит.

---

### 7. Запуск голосового ассистента

python voice_assistant.py

Далее:

1. Говорите в микрофон
2. Ассистент распознаёт речь
3. Находит ответ
4. Озвучивает его

## 📁 Структура проекта
```
voice_assistant/
│
├── rag/                # работа с базой знаний (RAG)
├── voice/              # STT и TTS
├── data/               # база знаний
├── db/                 # векторная база
├── utils/              # для работы с озвучкой
│
├── init_db.py             # инициализация БД
├── voice_assistant.py  # голосовой ассистент
```
